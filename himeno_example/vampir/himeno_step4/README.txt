Poisson FDM BMT の説明
                                                        2006年2月10日

◎配布ファイルに含まれているもの（3ファイル）
・himenoBMTxpr.f(ベンチマーク本体)
・param.h(問題サイズパラメータファイル)
・paramset.sh(パラメータファイル作成シェル)


◎コンパイルと実行方法
本プログラムはFortran77 + MPIで記述されています．
このため，MPIのライブラリとFortranコンパイラが必要です．
まず実行する前に問題サイズおよびプロセッサ数を
設定するためにparamset.sh を実行します．例えば，

% paramset.sh XL 4 4 4

この例では XL サイズで問題サイズの各次元を４つに
分割したパラメータを作成することになります．
つまりXLサイズの問題を 4x4x4=64 CPU で計算する
パラメータファイルができあがります．

そして，コンパイルおよび実行の例としては，

% mpif77 Poisson_FEM_BMT.F 
% mpirun -np 16 ./a.out 

のようになります．ここで，16 は使用するプロセッサ数を表しています．
